# レシート管理アプリ

## 概要
現金払いやコンビニ支払いのレシートが溜まり、手入力が負担だったため自分用に作成したWebアプリです。

## できること
- レシート画像をアップロードするだけでAIが自動読み取り
- 日付・店名・金額を自動抽出
- 読み取り結果を手動で修正可能
- 一覧で管理・確認
- CSV出力でデータをダウンロード

## 使った技術
- HTML / CSS / JavaScript
- Claude API（画像認識・AI読み取り）

## 使い方
1. HTMLファイルをブラウザで開く
2. 右上の「APIキー」ボタンからClaude APIキーを設定
3. レシート画像をアップロード
4. AIが自動で日付・店名・金額を読み取り
5. 確認して「保存」を押すだけ

## 注意事項
- Claude APIキーが必要です（https://console.anthropic.com で取得）
- データはブラウザのローカルストレージに保存されます
